# 扫码记录模块 使用说明

## 一、功能介绍
- **自动记录**：相机扫码、相册选图、长按识别等所有扫码结果自动保存。
- **智能分类**：自动识别支付码、群邀请、公众号等类型，显示对应图标。
- **便捷管理**：可复制、分享、重新打开任意记录；支持自定义备注、置顶、收藏。
- **归类规则**：设置链接前缀规则，匹配的记录自动归入独立分类。
- **快捷入口**：提供三种打开面板的方式（详见使用方法）。

---

## 二、使用方法

### 1. 打开扫码记录面板（三种方式）
- **① 主页加号菜单**  
  微信首页 → 右上角 **“+”** → 点击菜单中的 **“扫码记录”**。
- **② 聊天命令**  
  在任意聊天输入框发送 **“扫码记录”** 或 **“#扫码记录”**。
- **③ 设置按钮**  
  点击 **“设置”**按钮 → 直接显示记录面板。

### 2. 查看与操作记录
- **点击记录**：弹出详情弹窗，可查看完整内容。
- **长按内容区域**：自动复制链接，并提示“已复制”。
- 详情弹窗操作：
  - **复制链接**：复制到剪贴板。
  - **重新打开**：模拟扫码跳转结果页。
  - **分享给好友**：多选好友/群聊转发。
  - **删除**：移除该记录（不可恢复）。
- **快速重新打开**：直接点击记录左侧的**类型图标**（如“¥”“群”）。

### 3. 编辑记录（备注/置顶/收藏）
- 在记录列表中，点击**标题文字**（如“微信支付二维码”）。
- 可修改：
  - **自定义名称**：便于识别的备注。
  - **置顶**：开关后记录始终置顶。
  - **收藏**：开启后清空时保留。
- 点击 **“保存”** 生效。

### 4. 分类与自动归类
面板顶部标签页：
- **全部记录**：展示所有历史。
- **自动归类**：显示已匹配规则的分组记录，并可管理规则。
- **收藏**：仅展示收藏记录。

**添加归类规则**（在“自动归类”页）：
1. 点击 **“+ 添加规则”**。
2. 输入 **名称** 和 **链接前缀**（如 `https://mp.jtexpress.com/`）。
3. 保存后，匹配的记录自动归入该分类。

### 5. 清空记录
- 点击面板顶部的 **“清空记录”** → 弹出确认框。
- 默认**保留收藏**；如需全部清除，勾选“同时清空收藏内容”后再点“清空”。

### 6. 发送记录文本
- 在任意聊天发送 **“扫码记录发送”**，将最近 20 条记录以文本形式发送给当前聊天对象。

---

## 三、注意事项
- 记录上限 **200 条**，超出后自动删除最旧的非置顶/非收藏记录。
- 相同内容在 **1.5 秒内** 不会重复记录，避免冗余。
- 类型识别基于链接格式，个别特殊链接可能不准确。
- “重新打开”功能依赖当前微信版本，若失效请等待模块更新。
