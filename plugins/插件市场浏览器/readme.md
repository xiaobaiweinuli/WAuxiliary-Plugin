# 插件仓库

浏览、搜索并查看 WAuxiliary 插件仓库中的所有插件，支持自定义仓库和代理管理。

## 这是做什么的

- 展示官方插件仓库列表，自动标记已安装的插件
- 支持按名称或作者实时搜索筛选
- 支持添加自定义仓库（网络链接或本地文件）
- 支持管理 GitHub 加速代理，一键切换

## 怎么触发

- 点击该插件中的 **设置按钮** 打开。

- 插件会自动在微信主页面加号菜单中添加"插件市场"菜单项，可以点击按钮打开。

## 界面说明

打开后可以看到：

- **顶部**：插件总数和已安装数量，右侧有「⚙ 代理」和「管理」按钮
- **搜索框**：输入插件名称或作者名即时过滤列表
- **切换栏**：添加自定义仓库后会出现，点击可切换不同仓库
- **插件卡片**：显示名称、版本、作者，按钮颜色含义：
  - 🔵 蓝色「查看」— 未安装
  - 🔴 红色「已安装」— 已安装且是最新版
  - 🟠 橙色「更新」— 已安装但有新版本

## 自定义仓库

点击「管理」按钮：
- 可以看到已添加的自定义仓库列表，每条右侧有「删除」按钮
- 在底部表单填写名称和 JSON 链接后点「保存」即可添加
- 本地路径以 `/` 或 `file://` 开头，会自动识别为本地仓库
- 自定义仓库的 JSON 格式需与官方格式一致：`{"plugins":[...]}`

## 代理管理

点击「⚙ 代理」按钮：
- 列表中已有若干内置加速代理，当前使用的会高亮显示
- 点击某条代理即立即切换为当前使用
- 右侧「×」可删除，底部可添加新的代理地址
- 代理仅对官方仓库和原始文件下载生效，自定义仓库直接连接

## 注意事项

- 插件列表数据会缓存到本地，每次打开优先读取缓存并后台刷新
- 关闭插件时本地缓存会自动清理
- 菜单功能针对微信 `8.0.65` 版本开发，其他版本可能需要调整
- 菜单项 ID 使用 `9999`，如果与其他插件冲突，请修改 `CUSTOM_TYPE`
- 图标资源使用的是微信资源ID `2131692088`
